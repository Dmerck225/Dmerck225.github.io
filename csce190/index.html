Web Applications Final Website Front-End Code

This repository contains the frontend for the DeckMaster card management application, built using React.js. It provides a dynamic user interface for interacting with the backend, allowing users to manage card data, including viewing, adding, editing, and deleting cards. The application features a responsive design with user-friendly forms for inputting and displaying card information.

Web Applications Final Website Back-End Code

This repository contains the backend for the DeckMaster card management application. Built using Express.js, it provides RESTful APIs for managing card data, including adding, editing, deleting, and retrieving card information. The application supports handling image uploads for each card, allowing users to manage images and metadata such as card name, description, rarity, attack, defense, and abilities.
